The Science Behind It

The Baker-Baker Paradox

Told someone is a baker (occupation), you remember it easily because it connects to existing knowledge. Told their name is Baker, it's far harder. Names are semantically arbitrary, which is why learning them requires deliberate spaced practice.

Spaced Repetition Works

FSRS v4.5 models the forgetting curve with 17 optimised parameters. It shows you a face right before you'd forget it, making each review maximally effective. Studies show 99.6% superiority over older algorithms like SM-2.

Testing Effect

Actively recalling a name strengthens the memory far more than passively reviewing it. That's why this app progresses from easy recognition to challenging typed recall — each phase builds deeper memory traces.

Elaborative Encoding

Hearing a name pronounced, seeing it spelled, and typing it yourself creates multiple memory pathways. Each mode of engagement adds another retrieval cue, making the name easier to recall when you see the face in class.

2

3

4

Headshots are available from Week 1 each semester. This feature is only available for Course Coordinators. See the full UQ guide for details.
